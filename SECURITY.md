# Security Policy

## Supported Versions

This project currently supports the latest version available on the `master` branch.
Security updates are provided only for the latest release.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, please report it using one of the following methods:

- Use GitHub's **Private Vulnerability Reporting** feature
- Or contact the maintainer via GitHub

You can expect an initial response within **72 hours**.

If the vulnerability is accepted, we will work on a fix and release a security update as soon as possible.
